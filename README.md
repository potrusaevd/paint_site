# Prime_top_B2B_site

Краткое описание
- Prime_top_B2B_site - это веб-приложение для B2B-платформы, реализованное с использованием JavaScript (Node.js/Express), шаблонизатора EJS и CSS. Проект ориентирован на корпоративное использование и предоставляет готовую основу для витрины продуктов, управления контентом и интеграций.

Языковая компоновка репозитория
- JavaScript: 52.3%
- CSS: 29.1%
- EJS: 18.6%

Цели и назначение
- Быстрая развертка B2B-сайта с серверной отрисовкой страниц
- Простая расширяемая архитектура для подключения модулей учетных записей, каталога и интеграций
- Чистая и поддерживаемая структура фронтенда на EJS и CSS



Основные возможности (features)
- Серверная генерация страниц через EJS
- Маршрутизация и REST-эндпоинты на Express
- Статические ассеты: централизованная организация CSS и изображений
- Простая конфигурация окружения через .env
- Расширяемость: шаблоны и модули легко подключать и отключать
- Структура, ориентированная на масштабирование и тестирование

Структура репозитория (пример)
- /backend                   - основная серверная логика
- /backend/views             - шаблоны EJS
- /backend/package.json      - зависимости и команды
- /public                    - статические файлы (css, images, js)
- read.md                    - этот файл

Требования
- Node.js 16+ (рекомендуется актуальная LTS версия)
- npm 8+ или yarn
- Переменные окружения в .env (см. раздел Конфигурация)

Развертывание
- Рекомендуется использовать контейнеризацию (Docker) или PaaS-платформы
- Для production:
  - Установить NODE_ENV=production
  - Обеспечить переменные окружения
  - Настроить процесс-менеджер (pm2, systemd) или контейнер
